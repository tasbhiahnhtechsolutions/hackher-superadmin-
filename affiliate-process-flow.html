Affiliate Management System

Process Flow & Commission Structure — Based on meetings July 31 & August 4, 2025

Role Hierarchy & Relationships

Four-tier affiliate structure with cascading commission from discounted subscription price

Tier 0 — Platform Owner

Super Admin

Full visibility into all tiers. Sets commission percentages dynamically. Can assign/reassign affiliates to managers. Sees entire revenue breakdown. SA and SAM can adjust commissions, but SAM CANNOT adjust their own (only SA can).
Tier 1 — Oversight · 1% Commission

Super Admin Manager

Linked to multiple Managers (not affiliates directly). Monitors affiliate tree, cancellation reports, and payout status. Receives 1% of discounted price per subscription. SAM can set commission percentages for Managers and Affiliates but NOT their own.
Tier 2 — Management · 4% Commission

Manager

Onboards & manages a group of affiliates. Generates promo codes. Views only their own affiliates (not other managers'). Can see campaign analytics, download CSV reports. Manager CAN adjust affiliate commissions but NOT their own.
Tier 3 — Promotion · 10% Commission

Affiliate / Influencer

Logs in to see earnings, promo code performance, subscriber count. Can have multiple promo codes per campaign (Instagram, TikTok, etc.). Sees graphs of performance over time. Affiliates see which plan converted ($49 vs $99). Numbers based on Day 16+ customers only.
End User · Gets Discount

Customer

Uses influencer's promo code at signup. Gets a discount (e.g. 10–15% off subscription). Coupon only executes after trial → paid conversion (Day 15).
Key Rule: Managers see only their own affiliates Each manager can only view and manage their respective affiliates — not affiliates belonging to other managers. This prevents poaching and maintains trust. (Discussed Aug 4 meeting)
Key Rule: Trial users are not tracked until payment The coupon code does not execute during the 14-day trial. Affiliate attribution only happens on Day 15 when the customer converts to paid. Trial user counts are desired for marketing purposes but are separate from commission tracking. (Discussed Aug 4 meeting)
Commission Change Accountability Commission changes must be logged and timestamped for accountability every time SAM or Manager changes them.
Commission Cap (30% Rule) Total commissions + discounts must not exceed 30% of the full subscription price.

Privacy & Accountability Principle Each tier sees only what's relevant to their role. Managers cannot see other managers' affiliates to prevent poaching. Only Super Admin and SAM see the full financial picture. Commission percentages appear as read-only fields in promo code forms for lower tiers. All promo code and commission changes are tracked in the Change Logs section visible to SA, SAM, and Manager (each seeing their tier and below). SA can set all commissions; SAM can set Manager + Affiliate commissions but NOT their own; Manager can see Affiliate commission (read-only).
